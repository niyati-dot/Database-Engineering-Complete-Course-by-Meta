In this course you'll explore how to write database driven applications in Python by creating various types of clients that connect to MySQL databases using Python code and Python-related MySQL features and tools.

By the end of this course, you'll be able to:

  • Utilize Python code to create, populate and manipulate MySQL databases and tables

  • Access advanced functionality in MySQL using custom built Python clients

  • Develop working familiarity with advanced topics in MySQL

  • Apply the principles of advanced MySQL topics to problem solving using Python

  • Develop a working knowledge of the methods by which a MySQL database connects to the web via a Django API

  • Create a useful Python application capable of administration of a MySQL database

Module 1

In the first module, you'll receive an introduction to interacting with a MySQL database using Python. You'll discover what a database client is and how they're used in Meta.

The second lesson of this module focuses on how to establish a MySQL/Python connection. You'll learn how to install and set up the Jupyter notebook and related Python libraries. You'll then connect to a MySQL database using a Python client. Once you've established a connection, you'll learn how to create a table structure on your MySQL database using Python.

The final lesson in this module is about cursors. You'll explore how to work with cursors and MySQL, review different cursor subclasses and demonstrate your new skills in a lab environment.

By the end of this module, you'll be able to:

  • Make a connection to a MySQL database using Python

  • Create and define databases using Python

  • and process return sets with cursors

Module 2

The second module in this course teaches you how to perform queries in MySQL using Python.

In the first lesson, you'll explore how to perform create, read, update and delete (or CRUD) operations in a MySQL database using Python.

You'll first learn how to create and read records in a MySQL database with Python using the SELECT statements that you covered in previous courses. You'll then learn how to make use of UPDATE and DELETE statements in Python.

In the second lesson you'll discover how to perform more advanced queries in a MySQL database using Python. Examples of these more advanced queries include filtering and sorting data and performing JOIN operations.

By the end of this module, you'll be able to:

  • Select data from a MySQL database using Python,

  • Edit MySQL records using Python,

  • and retrieve filtered and sorted data from a MySQL database using Python.

Module 3

In the third module you'll learn about advanced database clients.

The first lesson focuses on implementing MySQL functions using Python. You'll review examples of MySQL functions from earlier courses, and then discover how to implement these using Python.

You'll also explore different kinds of datetime functions that can be used to query date and time data in a MySQL database.

In the second lesson of this module you'll look at some examples of stored procedures that you should already be familiar with from other courses. You'll find out how to create these stored procedures in Python and execute them on a MySQL database.

In both of these lessons, you'll demonstrate your new skills by implementing MySQL functions and stored procedures using Python in a lab environment.

In the final lesson of this module, you'll explore the topic of connection pools. You'll develop an understanding of the concept of database connection pooling, and how it works. You'll then review the Python/MySQL Connection Pool.

In this lesson's lab, you'll work directly with connection pools.

By the end of this module, you'll be able to:

  • Access and use MySQL functions with Python,

  • Access and use stored procedures with Python,

  • and explain how connection pools work.

Module 4

In the last module, you'll have an opportunity to recap what you learned, identify your strengths and target topics that you would like to revisit in this course. You'll also test your new skills in a lab environment and demonstrate your knowledge by undertaking a graded assessment.

By the end of this module you'll be able to:

  • Synthesize the skills you learned in this course to create a stored procedure for an example project

  • And reflect on the next steps in your learning journey.
