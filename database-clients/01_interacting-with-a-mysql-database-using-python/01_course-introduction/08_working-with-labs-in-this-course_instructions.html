In this course you'll demonstrate your knowledge of databases by completing lab exercises. Review the following guidelines for instructions around how to access and carry out these labs.

Open a lab

1. You can open a lab by clicking on the Open Lab button within the Exercise items.

2. Opening the lab redirects you to the VS Code editor with a notebook (.ipynb file).

3. If you don't see the notebook, then select the Navigate option and verify that you are in the /home/coder/project directory. The Jupyter Notebook Kernel is running within the labs using an extension of VS Code.

Open a Jupyter notebook

1. Double-click a Jupyter notebook from the Explorer menu to open it.

2. Opening a notebook reveals the instructions and code required to complete the lab.

Select the Kernel

1. Before you begin a lab, you need to select a Kernel. Click the Select Kernel button from the right-hand corner of the interface to bring up a list of kernels.

2. Select Kernel (Python 3.8.10 64-bit) from the list of options.

Execute prerequisites

Several labs contain a notebook named Prerequisites . The purpose of this notebook is to create the database and tables required for the lab. Execute this notebook before you begin the lab.

Execute a cell

Hover next to a code cell to reveal the execute button. Select this button to execute the code in the cell. You can also press SHIFT + ENTER .

Create a cell and add code

1. To insert a new cell in your Jupyter notebook, scroll down and hover over the active cell to reveal the Add Code Cell and Add Markdown Cell options.

2. Select the Add Code Cell option.

3. This action opens a new cell in which you can write your code.

4. To stop your code, your need to interrupt the kernel. Select the Interrupt option from the Kernel menu to stop your code.

Database credentials

Always use Username ="root" and Password = "" for database connection in connection settings.
