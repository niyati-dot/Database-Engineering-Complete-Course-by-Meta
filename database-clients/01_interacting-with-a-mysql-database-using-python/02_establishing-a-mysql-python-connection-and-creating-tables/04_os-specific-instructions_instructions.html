This reading provides installation instructions for Python, Jupyter notebook and MySQL Connector/Python API on operating systems other than Windows.

Installing Python

You have learned about installing the required software and setting up the environment for this course on a Windows operating system (OS). The installation steps on other operating systems are not that different.

The first step is to ensure that the most recent, stable version of Python is installed on your machine. You can explore the official Python website [ https://www.python.org/downloads/ ] and download the version that is suitable for your computer.

The command prompt [CMD] is called Terminal on Mac and other Linux operating systems. Access the Terminal, then type and execute the following code: 

python3 --version

This displays the version of Python installed on your machine.

Typically, Mac OS comes with the installed version of Python that could be 2.x.x. Type and execute the following code: 

python --version

You may see the default installation of Python 2.x.x

Installing the Jupyter notebook

Once you have confirmed that you have the latest stable version of Python on your machine, you need to install the IDE. In this course, you're using Jupyter notebook.

You should also be familiar with pip, which is a package manager for Python packages. Pip is used to install and manage packages or software that are not part of the Python standard library.

To install the Jupyter notebook, type the following code in the Terminal window: 

python3 -m pip install jupyter notebook

The above command installs the Jupyter IDE along with all of its dependencies.

Once the Jupyter notebook is installed, type the following code in the Terminal. 

meta$jupyter notebook

A new instance of the Jupyter notebook opens in your default web browser. Now, go to the New tab and select Python 3 (ipykernerl) .

Selecting Python 3 (ipykernerl) opens a new Jupyter notebook. You are now ready to code.

Coding with Jupyter

Type the following code to install the MySQL Connector/Python: 

!pip install mysql-connector-python

When you run this code, it installs the MySQL Connector/Python package (also called API or Client) along with all of its dependencies.

You are now ready to continue using your Mac or another Linux-based operating system in this course.
