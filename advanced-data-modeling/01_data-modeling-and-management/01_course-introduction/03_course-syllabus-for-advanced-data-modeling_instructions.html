In this course, you will develop a working knowledge and familiarity with advanced database concepts such as usage, modeling, automation, storage, optimization and administration.

To take this course, you must have completed the previous course on Database Clients. You must also be eager to continue your journey with coding.

The Professional Certificates create opportunities so that anyone, regardless of education, background or experience, can learn high-quality skills to land a high-growth career. There is no degree or experience required to get started.

Module 1

You'll begin the course with an introduction to the topic of advanced database modeling. You'll learn what database modeling is and discover how it's used at Meta.

You'll then explore database modeling in more detail by learning about different kinds of data models, and recap the topic of database normalization.

You'll also review different kinds of models that you can use to design your database.

In the second lesson of this module, you'll receive an introduction to MySQL Workbench, and discover that it's a unified visual tool for database modeling and management. You'll learn how MySQL Workbench is used to build data model diagrams using the forward engineering method, and how to reverse engineer data models.

By the end of this module, you'll be able to:

  • Describe the benefits of database modeling.

  • Differentiate three types of data models.

  • Design a data model by following a systematic data modeling approach.

  • Explore the difference between the different types of data models.

  • Review the principles of database normalization.

  • Explore how to manipulate and manage data using MySQL Workbench.

  • Explore how to model a database using MySQL Workbench.

  • Design a basic database model using WB visual data modeling tool.

  • And design a relational database model by using a systematic approach and applying the correct normalization levels.

Module 2

In the next module, you'll explore the topic of data warehousing. In this module, you'll learn how to architect a data warehouse and build a dimensional data model.

You'll begin with an overview of the concept of data warehousing, review the characteristics of a data warehouse, and learn about the different forms of data that a data warehouse works with.

You'll then explore the structure of a data warehouse and learn about the different components it's comprised of. You'll explore how these components control the flow of data from different sources, and then process and integrate this data so that users can perform data analysis.

In the second lesson of this module, you'll explore dimensional data modeling. You'll begin the lesson with an overview of the fundamentals of dimensional data modeling, and explore some examples.

You'll review the four key steps for creating a model, then undertake a lab exercise in which you'll create your own dimensional model.

By the end of this module you'll be able to:

  • Characterize the purpose and features of a data warehouse.

  • Explore the data warehouse architecture and explain how it works.

  • Elaborate on the role ETL process in data warehousing.

  • Explain the dimensional data modeling approach for data warehousing.

  • Explore the design of the star and snowflake schemas as used to build a data warehouse.

  • And explore how to create dimensional data modeling using the Four Steps approach.

Module 3

In the third module of this course, you'll explore data analytics in the context of data modeling and learn how to make use of analytics with data analysis software.

You'll recap key data analytics concepts from previous courses, and explore the different types of data and measurement scales you'll make use of during the analysis process.

Next, you'll learn about the topics of data mining and machine learning. You'll learn that data mining is the process of detecting patterns in data, and that machine learning is the process of teaching a computer how to learn.

You'll also review some examples of data mining models, and how they're used to process data.

You'll then learn about data visualization. You'll learn that, when visualizing your data, you must consider your audience and what information you need to present. You'll then need to choose an appropriate chart that best communicates your message.

Finally, you'll conclude this lesson with a discussion around what kind of data analytics reports you make use of.

In the final lesson of this module, you'll review the topic of data analytics, and learn how to make use of data analytics tools like Tableau.

As part of your introduction to Tableau, you'll learn what its key features are and how they help you to perform data analysis.

Finally, you'll undertake a lab exercise in which you'll perform data analysis in Tableau.

By the end of this module you'll be able to:

  • Describe the value and purpose of data analytics.

  • Explain the various types of data measurements.

  • Describe key concepts such as data mining and machine learning and other related concepts.

  • Explore the role of data visualization in the data analysis process.

  • Explore the role of advanced data analytics tools and evaluate how they can be used to understand the business performance.

  • Practice how to connect and prepare data in Tableau.

  • And create interactive dashboards in Tableau.

Module 4

In the fourth and final module, you'll undertake a graded assessment in which you'll complete your own data modeling project, making use of all the skills you learned in this course.

You'll then submit your model for peer review, and explore some next steps you can take to learn more about advanced data modeling.

By the end of this module you'll be able to:

  • Synthesize the skills you acquired in this course to model and analyze data for a given scenario.
