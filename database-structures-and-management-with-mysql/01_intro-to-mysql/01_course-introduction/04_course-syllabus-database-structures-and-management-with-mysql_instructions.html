To take this course, you must have completed the first course, Introduction to Databases. You must also be eager to continue your journey with coding.

Module 1

In module one, you'll receive an introduction to the course, including an overview of how Meta's database engineers use MySQL. You'll also receive tips on making the most of this course, along with a recap of key concepts from the first course.

You'll then explore how to filter data using logical operators such as AND, OR and NOT. You'll also learn about filtering data using the IN, BETWEEN and LIKE logical operators.

Next, you'll learn how to join tables using MySQL aliases. You'll review the different JOINS that can be used in MySQL and find out how to use a MySQL UNION operator.

In the final lesson in this module, you'll learn about the basics of grouping data. You'll investigate how the GROUP BY and HAVING clauses are used to group data and receive the opportunity to try grouping data yourself in an exercise.

B​y the end of this module you will be able to:

  • Filter data with logical operators.

  • Use various join types to query relevant data from multiple tables.

  • Use MySQL alias to improve the readability of the queries.

  • Group data from multiple tables using clauses and operators

Module 2

In the second module, you'll focus on updating databases and working with views. In the first lesson in this module, you'll learn how to insert and update data in a database using the MySQL REPLACE statement. You'll then explore the topic of constraints in lesson two.

In the third lesson, you'll discover how to change the structure of tables using the ALTER TABLE and COPY TABLE commands. The fourth lesson is on sub-queries, in which you'll learn how to execute a query within a query using comparison operators.

The last lesson in this module focuses on virtual tables or views. You'll learn how to create and work with views in MySQL and develop an understanding of the advantages that views offer.

B​y the end of this module you will be able to:

  • Update, insert and replace data in databases using MySQL REPLACE statement

  • Identify and work with constraints in MySQL databases

  • Change the structure of tables and copy table data using MySQL ALTER TABLE and COPY TABLE commands

  • Use subqueries and complex comparison operators in a MySQL database

  • Query data using MySQL CREATE VIEW command to create virtual tables

Module 3

The third module in this course revolves around functions and MySQL stored procedures. In the first lesson of this module, you'll learn how to make use of, and differentiate between, different kinds of functions. These include numeric, string and date functions. You'll also learn about comparison and control flow functions.

The second and final lesson in this module allows you to explore the concept of stored procedures in MySQL. You'll learn how to work with stored procedures and how you can use them with functions.

B​y the end of this module you will be able to:

  • Use different types of functions to query data

  • Create and deploy stored procedures in a MySQL database

Module 4

In the last module, you'll have an opportunity to recap what you learned and identify your strengths and target topics that you would like to revisit in this course.

B​y the end of this module you will be able to:

  • Demonstrate the skills and knowledge you have gained on this course.
