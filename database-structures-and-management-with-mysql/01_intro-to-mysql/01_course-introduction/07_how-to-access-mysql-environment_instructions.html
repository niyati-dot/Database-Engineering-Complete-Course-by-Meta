Accessing MySQL terminal on Coursera platform

You need access to the MySQL environment on the Coursera platform to complete the labs.

  • Click the open lab button on the course web page on Coursera.

  • You will be redirected to a new web page on the Coursera platform, as displayed.

  • Click on the Terminal tab.

  • Inside the terminal section, type the mysql keyword with a semicolon. 

mysql;

  • Click enter.

This will give you access to the MySQL environment, where you can start creating your database and tables. The image below confirms that MySQL is ready to use on the Coursera platform.
